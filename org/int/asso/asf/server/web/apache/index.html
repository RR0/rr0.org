Configuration

Elle est lue à chaque (re)démarrage depuis le fichier httpd.conf (HTTP Daemon), qui comprend des :

  • Directives :
    • DocumentRoot : répertoire racine des documents à servir.
    • Options : options, telles que :
      • FollowSymLinks : suivre les liens symboliques dans le système de fichiers (défaut) ;
      • Multiviews : accepter la négotiation de contenu (selon les en-têtes clients Accept, Accept-Language, Accept-Encoding, Accept-Charset) ;
      • Indexes : autoriser l'accès automatisé à aux indexs des répertoires ne contenant pas un fichier racine (index.html) ;
      • Includes : autoriser les inclusions côté serveur (SSI) via le mod_include.
    • AllowOverride : autoriser certaines directives dans le fichier .htaccess.
    • Order : .
    • Allow : .
    • LoadModule : charge un module mod_xxx au (re)démarrage du serveur.
    • Include : inclusion d'un autre fichier de configuration.
