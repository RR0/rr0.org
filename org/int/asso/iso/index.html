International Organization for Standardization : organisation internationale pour la standardisation.

Motivation

Favoriser le développement de normes internationales dans tous les domaines.

Analyse

L'ISO est une ONG fédèrant 140 organismes nationaux de normalisation, tels l'AFNOR en France ou l'ANSI aux USA.

Notes

  • Créée 1947
  • Basée à Genève

Exemples

Des exemples de normes ISO sont :

  • qualité
    • ISO 9000
  • langages
    • SQL1 (1987)
    • Ada
    • C++ (1998)
    • C (1999)
