Advanced Message Queuing Protocol : protocole avancé de files de messages.

Motivation

Définir un standard d'échange de messages entre applications.

Analyse

En particulier AMQP vise une interopérabilité maximale en garantissant l'échange de messages entre :

  • organisations différentes
  • moments différents (i.e. le destinataire n'a pas besoin d'être là au moment où le message est envoyé)
  • lieux différents, y compris en traversant des réseaux de mauvaise qualité
  • technologies différentes, garantissant notamment l'échange entre des plates-formes différentes (i.e. chaque application est libre de choisir sa plate-forme/ses technologies)

Notes

  • Standard de OASIS
