Open Neural Network Exchange : échange ouvert de réseaux de neurones.

Motivation

Permettre d'échanger des modèles entre différents frameworks d'apprentissage profond.

Analyse

Standard de représentation des modèles de ML définissant :

  • un ensemble d'opérateurs
  • un format de fichier

Conception

ONNX définit :

  • des types de données standards
  • des opérateurs prédéfinis
  • un modèle de graphe de calcul acyclique, constitué de :
    • noeuds ayant :
      • n entrées ou plus
      • n sortie ou plus
      • un opérateur à appeler
    • métadonnées décrivant objectif, auteur, etc.

Exemples

Permet d'échanger des modèles entre les frameworks :

  • CNTK
  • Caffe2
  • MXNet
  • PyTorch
