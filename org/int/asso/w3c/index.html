World Wide Web Consortium : consortium de la toile mondiale.

But

Standardiser et promouvoir la compatibilité des technologies du Web

Analyse

Le W3C n'émet pas des normes au sens européen, mais des recommandations à valeur de standards industriels.

Exemples

Le W3C normalise par exemple :

  • HTML
  • XHTML
  • XML
  • RDF
  • CSS
  • PNG
  • SVG
  • SOAP

Notes

  • Organisme à but non-lucratif
  • fondé en octobre 1994
  • Parmi les membres du W3C on compte les plus grands acteurs du Web actuels tels que Microsoft, IBM, Opera, Google, Apple, etc.
