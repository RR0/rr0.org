Web Hypertext Application Technology Working Group.

But

Standardiser et promouvoir la compatibilité des technologies du Web

Analyse

Le n'émet pas des normes au sens européen, mais des recommandations à valeur de standards industriels.

Exemples

normalise par exemple :

  • HTML
  • XHTML
  • XML
  • RDF
  • CSS
  • PNG
  • SVG
  • SOAP

Notes

  • Organisme à but non-lucratif
  • fondé en 2004
  • Parmi les membres, on compte les plus grands acteurs du Web actuels tels que Microsoft, IBM, Opera, Google, Apple, etc.
