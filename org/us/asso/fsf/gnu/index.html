Gnu's Not Unix : Gnu N'est pas Unix.

Besoin

Développer un OS :

  • semblable à Unix
  • libre
  • complet

Analyse

Les produits GNU sont "libres", c'est-à-dire :

  • gratuits
  • redistribuables
  • modifiables

Cette liberté est stipulée dans la GPL.

Ils sont généralement livré entourant un noyau (kernel), le plus souvent Linux.
