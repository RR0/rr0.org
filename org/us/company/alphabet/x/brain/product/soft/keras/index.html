API de haut niveau

Conception

Keras est une API s'appuyant sur des bibliothèques ML existantes, nommées "backends"

Implémentation

Keras est développé en Python, mais propose une bibliothèque JavaScript pour exécuter ses modèles.

Exemples

Des exemples de backends Keras sont :

  • Tensorflow
  • Theano
  • MCT
  • PlaidML
