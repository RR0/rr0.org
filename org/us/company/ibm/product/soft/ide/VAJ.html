VisualAge for Java.

Besoin

Développer des applications Java, en particulier pour WAS 3.x.

Conception

VAJ stocke ses repository dans un fichier .dat.

Notes

  • Incontournable pour le développement sur WAS < 4.
  • VAJ est la déclinaison Java de l'outil VisualAge, existant à l'origine pour SmallTalk, puis pour C++.
