Besoin

Analyse interactive.

Analyse

Un notebook est constitué d'une séquence de cellules (cells) de type :

  • code à exécuter dans différents langages (Python, etc.)
  • texte (markup) à lire

Chaque cellule peut être en mode :

  • édition pour rédiger du code ou du texte riche
  • commande pour exécuter du code.

Conception

Un notebook peut être sauvegardé dans un fichier au format spécifique.

Implémentation

Raccourcis
Contexte Raccourci Description
Edition Esc Passe en mode commande
Ctrl + Shift + - Scinde la cellule à l'endroit du curseur
Commande Enter Passe en mode édition
Haut Remonter à cellule précédente
Bas Descendre à la cellule suivante
Shift + Bas Sélectionner des cellules vers le bas
Shift + Haut Sélectionner des cellules vers le haut
A Insérer une cellule au-dessus (Above)
B Insérer une cellule en-dessous (Below)
M Transformer la cellule en texte (Markdown)
Y Transformer la cellule en code
DD Supprimer la cellule
Z Annuler la supression de cellule
Clic à gauche de cellules Sélectionne la plage de cellules
Cellules sélectionnées M Fusionner (Merge) les cellules
