Conception

Node.js intègre :

  • le moteur d'exécution V8
  • des bibliothèques ("modules") :
    • assert
    • buffer
    • child_process
    • cluster
    • crypto
    • dgram
    • dns
    • events
    • fs
    • http
    • https
    • net
    • os
    • path
    • punycode
    • querystring
    • readline
    • repl
    • string_decoder
    • tls
    • tty
    • url
    • util
    • vm
    • zlib

D'autres module peuvent être recherchés, téléchargés et installés via son utilitaire npm.

Implémentation

Les versions sont numérotées 0.x.y.

À partir de 2015-09, les versions parait une nouvelle version commençant à 4.x.y
