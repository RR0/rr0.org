Bibliothèque pour Node.

Un module Node.js est typiquement implémenté dans un fichier .js à part, contenant une déclaration des fonctions exportées :

  • soit via export.maFonction = function(params) { ... }
  • soit via module.exports = function(constructorParams) {
    var func1 = function (params1) { ... }
    var func2 = function (params2) { ... }
    return {
    api1: func1,
    api2: func2
    }
    }

Un module peut être importé via var mymodule = require('mymodulename');
