Installation

Locale

Globale

Lorsqu'un module a une vocation utilitaire et n'est pas spécifique à un projet, on utilise une installation globale qui, outre le fait de placer le module dans un répertoire global de modules, produit un exécutable spécifique à la plate-forme d'installation pour une invocation plus aisée.

npm install -g someModule
