Besoin

Fournir un navigateur Web Open Source respectant les standards.

Analyse

Mozilla est une plate-forme intégrant différents composants :

  • Navigateur Web SeaMonkey, incluant :
    • le moteur de rendu NGLayout/Gecko
  • intégration Java
    • OJI
    • Blackwood
  • JavaScript via
    • SpiderMonkey (implémentation C)
    • Rhino (implémentation Java)

Notes

  • Issu de la publication du code source de Netscape Navigator en 1998.
  • Versions équivalentes produites par Netscape (Netscape 6 et ultérieurs).
  • Licences NPL/MPL/GPL/LGPL

FUD

  • Mozilla n'est pas stable : cela a été vrai mais ne l'est plus depuis Mozilla 1.0
  • Mozilla est trop lourd (avec navigateur, client mail, etc.) : mozilla assemble effectivement ces composants mais ceux-ci sont également disponibles séparément (navigateur Firebird ou Camino, client mail Thunderbird)
  • Mozilla ne respecte pas les derniers standards : au contraire c'est un des objectifs principaux de Mozilla.
