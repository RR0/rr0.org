Microsoft CogNitive ToolKit (CNTK) : boîte-à-outils cognitive de Microsoft.

Besoin

Bibliothèque logicielle pour l'apprentissage profond.

Analyse

Peut être utilisé comme :

  • bibliothèque dans des programmes
  • outil via son propre langage de description de modèle (BrainScript).

Supporte le format ONNX.

Conception

CNTK décrit les RN comme une série d'étapes de calcul à travers un graphe orienté.

Implémentation

Peut être inclus comme bibliothèque dans des logiciels écrits en :

  • Python
  • C++
  • C#

L'évaluation de modèle CNTK peut aussi être utilisée depuis des programmes Java.

Notes

  • Utilisable comme backend Keras
  • Fonctionne sur des plates-formes 64 bits (Windows ou Linux)
