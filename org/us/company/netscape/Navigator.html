Besoin

Navigateur Web de Netscape.

Implémentation

Sécurité

Netscape Navigator sait télécharger des certificats X.509 V1, V2 ou V3 via HTTP, moyennant la spécification du type MIME approprié dans l'en-tête HTTP :

  • application/x-x509-ca-cert : certificat identifiant une Autorité de Certifcation.
  • application/x-x509-email-cert : certificat d'un autre utilisateur utilisable par S/MIME
  • application/x-x509-user-cert : certificat appartenant à l'utilisateur du navigateur Netscape (la clé privée correspondant au certificat doit de trouver dans la base de clé locale de Netscape).

Ces certificats peuvent être codés en formats :

  • Binaires
    • DER
    • PKCS#7
      • via le champ certificates son objet SignedData (le contenu et la signature de l'objet lui-même sont ignorés). Dans des versions futures, crls sera utilisé.
      • via la structure ContentInfo (contentType contient netscape-cert-sequence, et content une séquence de certificats).
  • Texte codé base64 (RFC 1113)
    • selon le format -----BEGIN CERTIFICATE----- <données du certificat> -----END CERTIFICATE----
