Besoin

Vendre des logiciels.

Notes

  • Créé en 1994 par Jim Clark, ex-fondateur de Silicon Graphics.
  • Vends les logiciels :
    • Navigateur Web : Navigator/Communicator via lequel elle a créé ou contribué à de nombreuses technologies cruciales de l'Internet comme :
      • HTML
      • JavaScript
      • les cookies
      • SSL
    • Serveurs
      • Web : Enterprise Server (NES)
      • Annuaire : Directory Server
      • Commerce : Merchant System (1995), basé sur l'interface Netscape Navigator, pour du commerce électronique en ligne sur Internet.
      • Applicatifs : iPlanet

Notes

  • Netscape a perdu son avance sur le marché des navigateurs Web suite à la sortie de MSIE livré avec Windows.
  • Racheté par AOL.
