Write Once Run Anywhere : écrivez une fois et exécutez n'importe où.

But

Portabilité d'une application sur diverses plates-formes.

Analyse

Le WORA permet de porter une application dans son format exécutable. C'est à dire :

  • sans modifier le code source
  • sans recompiler

Conception

La technique généralement utilisée pour réaliser le WORA est l'exécution sur une plate-forme indépendante de la plate-forme physique et de l'OS.

Limitations

  • Il faut implémenter la plate-forme virtuelle sur chaque plate-forme physique/OS.

Exemples

Le WORA est une qualité de :

  • Java et sa JVM
