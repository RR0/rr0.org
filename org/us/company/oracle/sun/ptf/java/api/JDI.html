Java Debug Interface : Interface de debugging en Java.

Besoin

Fournir une API simplifiant la manipulation d'échanges JDWP.

Analyse

JDI :

  • formate et décode les données à envoyer sur le réseau
  • bufferise les requêtes
  • cache des données
  • gère les connections
