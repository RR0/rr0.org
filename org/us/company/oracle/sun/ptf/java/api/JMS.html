Java Messaging Service : Service de Messagerie Java.

Motivation

  • Définir une API Java standard pour utiliser des messageries asynchrones (MOM) entre applications.

Structure

JMS offre une SPI pour l'implémenter.

Caractéristiques

  • Intégré à J2EE [1.3+]

Exemples

Des exemples d'implantations de JMS sont :

  • WebLogic Server
  • Sonic MQ
  • IBM MQSeries
  • ObjectWeb JORAM
