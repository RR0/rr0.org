Java Advanced Imaging : imagerie Java avancée.

Besoin

Fournir une API Java standard pour l'affichage et la manipulation d'images.

Analyse

JAI permet :

  • Réglage du contraste
  • Découpage
  • Redimensionnement
  • Transformations géométriques
  • Traitement des fréquences
  • La lecture/écriture de formats d'image (dépendant de Java 2, utiliser Jimi si JRE 1.1.x)

Exemples

Une telle API est utile dans des domaines comme :

  • la défense et le renseignement
  • Le traitement d'images géospatiales
  • Le traitement d'images documentaires
  • la bio-informatique
  • La recherche appliquée
  • La photographie numérique
  • L'enseignement et la revente à distance
