Service Data Objects : Objets de Données pour les Services.

Besoin

Simplifier et unifier la manière dont les applications manipulent les données.

Conception

Un client SDO interroge un service de médiateur de données (data mediator service) pour récupérer/créer, lire/modifier puis envoyer des graphes déconnectés d'objets de données vers leur source de donnée. Le médiateur de données permet également d'obtenir des métadonnées.

Notes

  • Créé en 2003.
  • Initialement framework de IGS, puis proposé comme standard par IBM et BEA.
  • Diverses implémentations de solutions de persistance existantes (JDO, Hibernate, etc.) peuvent fournir un service de médiateur de données.
