Besoin

Manipuler des scènes graphiques en 3 dimensions en Java

Conception

Java 3D est une API d'extension standard au-dessus d'implémentation natives pour tirer parti des accélérations 3D (OS, cartes graphiques).

Exemples

Des implémentations Java 3D existent pour :

  • Windows
    • Direct X
    • OpenGL
  • Solaris/SPARC
  • IBM AIX
  • HP-UX
  • Blackdown Linux
  • SGI IRIX
