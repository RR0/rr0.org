Java Web Services Developer Pack : pack Java pour développeurs de services Web.

Besoin

Offrir un ensemble d'API et d'outils pour le développement d'applications Java utilisant les services Web.

Conception

Le JWSDP regroupe un ensemble d'API Java optionnelles comprenant :

  • le Java XML Pack, un ensemble d'API Java optionnelles directement liées à XML.
  • la JSTL
  • des outils de développement et test :
    • le Java WSDP Registry Server
    • Web Application Deployment Tool
    • Ant Build Tool
    • Apache Tomcat Web container

Notes

  • Nécessaire pour JSF.
  • Installable sur J2EE RI
