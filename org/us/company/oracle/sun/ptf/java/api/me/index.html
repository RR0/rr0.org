Java 2 Micro Edition : édition micro de Java 2.

Besoin

Fournir une plate-forme d'exécution des applications Java pour :

  • les appareils connectés : PDAs sophistiqués (PocketPC), appareils embarqués (dans des véhicules), les boîtiers (décodeurs, etc.)
  • les appareils mobiles : téléphones mobiles ou PDAs légers et déconnectés

Conception

Une architecture J2ME est composée de :

  • une configuration : un machine virtuelle et un ensemble de librairies adaptées pour la plate-forme visée
  • des profils : un ensemble d'API de haut niveau
  • des packages optionnels
