Appliquette : petite application.

Besoin

Exécuter du code Java dans une page Web.

Conception

Une applet est insérée dans une page Web comme un plugin.

Implémentation

Aujourd'hui la balise <applet> est obsolète. Elle doit être remplacée par une balise de type <object>.

Exemples

Un exemple de code HTML intégrant une applet est :

<object codetype="application/java" classid="java:Bubbles.class" width="270" height="200">
Exemple d'applet.
</object>

Un autre exemple de code HTML intégrant une applet est :

<object codetype="application/java" classid="Bubbles" width="270" height="200">
<param name="snd" value="Hello.au|Welcome.au">
Exemple d'applet.
</object>

Un exemple obsolète de code HTML intégrant une applet est :

<applet code="net.javarome.presentation.SlideShowApplet" width="270" height="200" hspace="20" vspace="20" align="right">
Exemple d'applet.
</applet>

Limitations

  • Déploiement sujet aux variations d'implémentation de J2SE dans les différents navigateurs.
    • MSIE n'est livré qu'avec J2SE 1.1 par défaut
    • Nécessité d'utiliser le JPI
