Java Metadata Interface : interface de métadonnées Java.

Besoin

Permettre la découverte, recherche, l'accès et la manipulation de métadonnées, à la conception comme à l'exécution.

Analyse

JMI est la version Java du MOF 1.4.

Conception

JMI est un ensemble d'interfaces

Il utilise XMI pour l'échange de métadonnées.

Implémentation

J2SE 1.3+

Exemples

Des exemples d'implémentations de JMI sont :

  • L'implémentation de référence (RI) de Unisys
  • le Netbeans Metadata Repository (OSS)
  • le SAP NetWeaver
