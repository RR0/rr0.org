Java 2 Standard Edition : édition standard de Java 2.

Anciennement JRE (Java Runtime Environment).

Besoin

Fournir une plate-forme d'exécution des applications Java.

Analyse

J2SE est la plate-forme Java de base ou "standard".

Elle inclut :

  • un ensemble d'API standards
  • une configuration comprenant :
    • la JVM en cours (HotSpot client ou serveur)
    • une politique de sécurité (fichier .java.policy)
    • des certificats de CA (fichier cacerts)
