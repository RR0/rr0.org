Java Network Launching Protocol : protocole Java de démarrage réseau.

Conception

JNLP est une extension de HTTP.

Session JNLP
HTTP
Transport
TCP
Réseau
IP

JNLP ajoute deux nouveaux types MIME :

  • application/x-java-jnlp-file .jnlp
  • application/x-java-archive-diff .jardiff

Exemple

<jnlp codebase="...
