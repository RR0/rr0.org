Java Web Start : démarrage Java depuis le Web.

Désignant plus rarement feu le Java Web Server.

Besoin

Permettre une installation automatisée d'applications Java lourdes sur le poste client.

Analyse

Détection de la non mise à jour et à téléchargement du delta.

Conception

JWS utilise le protocole JNLP.
