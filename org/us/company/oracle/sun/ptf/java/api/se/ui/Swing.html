Adaptive Software Development : développement logiciel adaptatif.

Besoin

Fournir pour le développement logiciel un environnement adaptatif plutôt que prédictif.

Analyse

ASD est une méthode agile de développement logiciel. Elle considère que l'imprévu est inévitable, et que l'on peut en tirer parti pour atteindre de meilleures solutions, qui n'auraient pas été envisagées a priori.

Elle comporte 3 phases (mais non linéaires et parallèles) :

  1. spéculation
  2. collaboration
  3. apprentissage

Elle s'attache à discerner tous les acteurs :

  • clients
  • développeurs
  • management
