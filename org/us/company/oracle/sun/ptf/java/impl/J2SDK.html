Java 2 SDK.

Ex-JDK (Java Development Kit).

Besoin

Fournir une plate-forme de développement et d'exécution.

Analyse

Le J2SDK comprend des plates-formes :

  • d'exécution (J2SE) capable d'exécuter des applications compilées en bytecode Java.
  • de développement composée de divers outils :
    • javac (Java Compiler)
    • javadoc (Java DOCumentation)
    • jar (Java ARchiver)
    • jarsigner pour signer une archive Java
    • rmic (RMI Compiler)
    • rmiregistry
    • keytool pour gérer une base de clés et certificats
    • policytool pour gérer une politique de sécurité
    • jdb (Java DeBugger)
    • idlj (IDL to Java)
