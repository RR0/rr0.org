Java Debug Wire Protocol : Protocole de communication de debugging en Java.

Besoin

Fournir un protocole réseau portable permettant la mise-au-point à distance (remote debugging).

Analyse

JDWP définit le format des informations et des requêtes transférées entre le processus débogué et l'interface de debugging.

Conception

JDWP fait partie de la JPDA.

Notes

  • JDWP peut être exploité directement par des applications (par exemple si celles-ci ne sont pas en Java), mais le cas le plus courant (car le plus pratique) depuis des applications Java reste son utilisation à travers la JDI.
