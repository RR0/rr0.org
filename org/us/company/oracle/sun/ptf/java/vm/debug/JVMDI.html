JVM Debug Interface : Interface de debugging des JVMs.

Besoin

Fournir une API simplifiant les requêtes de debugging dans une JVM.

Notes

  • Remplacé par JVM TI.
  • JVMDI peut être utilisé directement par des applications (par exemple si celles-ci ne sont pas en Java ou veulent fournir du debugging mixte Java + autre), mais le cas le plus courant (car le plus pratique) reste son utilisation à travers la JDI ou JDWP.
