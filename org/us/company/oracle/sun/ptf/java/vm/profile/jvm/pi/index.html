JVM Profiler Interface : Interface d'optimisation (profiling) pour les JVMs.

Besoin

Fournir une API permettant l'interaction d'outil d'optimisation (profiling) avec une JVM.

Analyse

JVMPI permet :

  • d'envoyer des ordres depuis une application de profiling vers une JVM.
  • des événements venus de la JVM.
