Au sens strict l'énergie n'est pas "produite" à partir de rien (ex-nihilo) mais convertie à partir d'une autre énergie dite "primaire". De l'énergie électrique sera donc obtenue par le biais d'une :

  • énergie mécanique :
    • d'une dynamo/un générateur, lui-même mû par une énergie thermique ;
    • d'un courant (éolien, hydraulique)
  • énergie photovoltaïque issue du rayonnement solaire

On peut également classer les énergies leur origine :

  • naturelle : renouvelables mais non pilotables ;
  • artificielle (énergies fossiles, nucléaire) : pilotables mais généralement basées sur l'exploitation de stocks de combustible.

Impact

Chaque mode de production d'énergie (électrique) a un impact :

  • écologique (pollution)
  • économique (coût financier, emplois)
  • pratique (pilotable ou non)
