La pollution est une conséquence nocive pour l'environnement et/ou la santé d'une activité humaine.

Des exemples sont les pollutions :

  • atmosphérique :
    • le réchauffement climatique (suite à l'accumulation de GES émis, notamment par des techniques de production d'énergie)
    • CFC creusant le trou dans la couche d'ozone
    • émission de particules fines, cancérigènes (par les véhicules terrestres ou aériens, les usines)
  • terrestre
    • la production de déchets (et leur taux de recyclage) induites par la consommation et certaines techniques de production d'énergie
    • substances nocives (cancérigènes, perturbateurs endocriniens) dans les aliments ingérés (par les animaux, les humains)
  • énergétique : la quantité de matériau nécessaire à la production d'une même quantité d'énergie
