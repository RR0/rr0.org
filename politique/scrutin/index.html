Scrutin

Ensemble des opérations constituant une élection.

Analyse

Un mode de scrutin est caractéristé par :

  • sa représentation
    • majoritaire : les élus sont les premiers au classement
    • proportionnelle : le nombre d'élus dépend du nombre de suffrages pour chacun
  • son nombre d'élus/sièges
    • uninominal (élection d'un candidat)
    • plurinominal (éléction d'une liste)
  • son nombre de tours
    • 1 tour
    • 2 tours

Exemples

Des exemples d'élections appliquant des mode de scrutins sont :

  • majoritaire
    • élections sénatoriales pour les petits départements
    • uninominal
      • élections législatives : un député de l'Assemblée Nationale est élu dans chaque circonscription définie par le « découpage » électoral.
  • proportionnelle :
    • élections sénatoriales pour les grands départements
