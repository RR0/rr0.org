Les Rencontres Rapprochées du 1er type (Close Encounter of the 1st kind ou CE1) représentent le type minimum de Rencontres Rapprochées.

Dans ce type d'observation la possibilité de méprise est théoriquement réduite au minimum, l'ovni se trouvant suffisamment près pour être dans le champ de vision immédiat du témoin.

Les RR1 impliquent généralement des formes diverses (disques, ovales, sphères, parfois dômes) en sustentation dans l'air et en rotation. On constate aussi souvent des accélérations rapides en des ascensions ou descentes très raides. Les observations sont presque toujours silencieuses, malgré la proximité de l'observateur.

Toutefois dans ce type d'observation l'ovni n'interragit pas avec l'environnement, comme dans les RR2.
