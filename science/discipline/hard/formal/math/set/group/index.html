Analyse

Un groupe mathématique est un ensemble ayant :

  • une loi de composition interne qui permet de trouver un élément à partir de deux autres
  • un élément neutre (zéro typiquement)
  • des éléments symmétriques (-1 et +1 par exemple)

Exemples

  • Groupes des entiers
    • naturels : $\mathbb{N}$
    • relatifs : $\mathbb{Z}$
  • Groupe unitaire
