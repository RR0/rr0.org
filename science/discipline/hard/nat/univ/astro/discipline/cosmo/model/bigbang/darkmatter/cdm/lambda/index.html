Lambda - Cold Dark Matter est un modèle CDM paramétré par une constante cosmologique `Λ` (lambda) associée à l'énergie sombre.

Comme le modèle SCDM qu'il remplace, il est basé sur la RG.

Le modèle ΛCDM minimal est défini par les paramètres suivants (aux effets indépendants) :

  • `A_s` : amplitude globale du spectre primordial
  • `n_{s}` : indice spectral des perturbations primordiales scalaires de densité
  • `\omega _{b}` : densité de baryons
  • `\omega _{m}` : densité de matière non-relativiste, avec `\omega _{m}=\Omega _{m}h^{2}`
  • `\Omega _{\Lambda}` : densité fractionnelle de constante cosmologique
  • `\tau` : épaisseur ou profondeur optique de la réionisation
