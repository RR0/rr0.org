Application Programming Interface : interface de programmation d'application.

Besoin

Fournir un moyen de communication entre deux sous-systèmes au niveau langage.

Principes

Une API une est interface d'implémentation. A ce titre est elle spécifique à un langage.

Exemples

Des exemples d'API sont :

  • l'API standard de J2SE
  • BEA JOLT pour accéder à Tuxedo depuis Java
  • JDBC pour accéder aux SGBDR depuis Java
  • l'API de Log4J pour effectuer des traces (logs)
  • l'API des sockets BSD en C

Notes

  • A ne pas confondre avec un canevas (framework) que le développeur utilise comme une base d'implémentation à compléter.
