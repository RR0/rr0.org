Besoin

Logiciel développé.

Conception

Une application s'exécute sur une plate-forme sous la forme de 1 ou plusieurs processus.

Exemples

Des exemples d'applications sont :

  • une application Web s'exécutant dans un serveur applicatif
  • une application indépendante s'exécutant sur une plate-forme Java
  • une application native s'exécutant sur Windows 2000
  • une application native s'exécutant sur Solaris 8
