Connected Device Configuration : configuration connectée pour périphérique.

Besoin

Définir une configuration pour J2ME.

Analyse

CLDC est adapté pour :

  • les connexions réseau intermittentes
  • les microprocesseurs peu puissants
  • la place mémoire limitée

Conception

CDC
Machine virtuelle JVM
Configuration CDC
Profils FP
Réseau FB
GUI PBP GUI basique
PP (Ex-PersonalJava) AWT, Applets

Déploiement

Les plates-formes matérielles hébergeant CDC doivent avoir :

  • un microprocesseur 32 bits
  • de 2 Mo de RAM

Exemples

Des exemples de plates-formes pouvant héberger CDC sont :

  • PDAs sophistiqués (PocketPC)
  • appareils embarqués (dans des véhicules)
  • boîtiers (décodeurs, etc.)
