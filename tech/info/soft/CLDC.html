Connected Limited Device Configuration : configuration connectée pour périphérique limité.

Besoin

Définir la Configuration minimale pour J2ME.

Analyse

CLDC est adapté pour :

  • les connexions réseau intermittentes
  • les microprocesseurs peu puissants
  • la place mémoire limitée

Conception

CLDC
Machine virtuelle KVM
Configuration CLDC
Profils MIDP

Déploiement

Les plates-formes matérielles hébergeant J2ME doivent avoir :

  • un microprocesseur 16 ou 32 bits
  • de 128 Ko à 512 Ko de RAM

Exemples

Des exemples de plates-formes pouvant héberger CLDC sont :

  • téléphones mobiles
  • PDAs légers et déconnectés
