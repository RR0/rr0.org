Connection.

Besoin

Maintenir un flux d'échange de données.

Analyse

Une connexion fournit des flux en :

  • écriture
  • lecture

Conception

Les connexions utilisent souvent TCP/IP comme protocole de transport des données.

Implémentation

Les connexions sont souvent manipulée via une API de prises (sockets).

Exemples

Des exemples de connexions sont :

  • une connexion entre une application et un SGBDR
    • sur laquelle un certain nombre de requêtes peuvent être émises en parallèle
  • une connexion HTTP entre un navigateur Web et un serveur Web
  • une connexion SSL

Notes

  • Les connexions sont souvent mutualisées (pooled)
