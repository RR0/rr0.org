Graphical User Interface : interface utilisateur graphique.

Besoin

Fournir une IHM graphique.

Analyse

Les IHM graphiques sont des assemblages de composants graphiques (widgets) tels que :

  • menus déroulants
  • zone de saisie
  • fenêtres déplaçable, redimensionnable, minimisables et maximisables
  • boutons
    • radio pour les choix uniques
    • case à cocher (checkbox) pour les choix multiples

Ces IHM sont utilisées via une souris et le clavier.

Exemples

Des exemples de GUI sont :

  • Web
    • les pages HTML
  • Java
    • l'IHM de AWT
    • les IHM de Swing
  • l'IHM de Windows
  • les Presentation Manager de Linux
  • l'IHM sur les Apple

Notes

  • Les systèmes de fenêtrage et menus déroulants furent inventés dans les laboratoires Xerox et appliqués sur machines Apple.
