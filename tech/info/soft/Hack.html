  • modifier des programmes
  • modifier de l'information

Analyse

Un hacker est typiquement une personne totalement immergée dans la technologie et la programmation informatiques, et qui aime examiner le code d'OS ou de programmes pour voir comment ils fonctionnent.

Ces personnes utilisent alors leur expertise informatique à des fins illicites pour obtenir un accès à un système informatique sans permission.

Conception

Les techniques utilisées peuvent être :

  • l'installation de portes dérobées (backdoors)
  • les virus
  • les chevaux de troies (trojan horses)
