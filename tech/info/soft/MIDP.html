M. I. D. P.

Mobile Information Device Profile : profil pour appareil d'information mobile.

Besoin

Fournir un ensemble d'API Java pour les téléphones mobiles et PDA simples.

Analyse

MIDP fournit des API pour :

  • la GUI
  • la connectivité réseau
  • le stockage local de données
  • la supervision d'applications

Conception

Le profil MIDP peut être ajouté sur une configuration CLDC.
