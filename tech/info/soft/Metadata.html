Méta-données : données sur des données.

Besoin

Décrire les données pour pouvoir les interpréter.

Analyse

Les métadonnées fournissent des données sur les données (leur structure ou leur signification/sémantique).

Conception

JMI définit un standard de métadonnées Java.

Exemples

Des exemples de métadonnées sont :

  • un schéma de base de données
  • une DTD ou un Schéma XML
  • Les métadonnées JDO, décrivant les classes et champs persistants
  • les modèles UML
  • XMI définit un format d'échange pour la description de modèles UML
