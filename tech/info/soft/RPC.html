Remote Procedure Call : appel de procédure à distance.

Besoin

Donner un ordre à un processus distant de manière synchrone.

Analyse

Le RPC est un appel de procédure (il n'est pas OO et ne s'exerce donc pas sur un objet distant).

Exemples

Des exemples de protocoles de types RPC sont :

  • ONC RPC
  • CORBA
  • les services Web
    • XML-RPC
    • JAX-RPC

Voir

  • RFC 1050 - Remote Procedure Call protocol specification
    • RFC 1831 - Remote Procedure Call protocol specification version 2
