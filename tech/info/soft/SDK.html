Software Development Kit : Kit de développement de logiciel.

Besoin

Fournir des outils pour développer avec un langage, une API (fournissant un service, permettant de s'interfacer avec un produit)

Analyse

Un SDK est composé d'outils comme :

  • un compilateur
  • un debogueur
  • documentation (générale, de l'API)
  • un générateur de documentation de code

Exemples

Des exemples de SDK sont :

  • J2SDK (javac, jdb, javadoc, keytool)
  • J2SDKEE (deploytool, realmtool)
