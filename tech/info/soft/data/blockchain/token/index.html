Jeton.

Motivation

Représenter un actif numérique non duplicable.

Analyse

On distingue en fait :

  • les jetons natifs (native tokens) ou "coins" qui sont les unités de valeur d'une blockchain donnée (Bitcoin, Etherum, etc.)
  • les autres jetons (pour lesquels on réserve généralement le terme de jeton/token) qui se base sur les jetons natifs. Par exemple un jeton COSS est basé sur le "coin" de la blockchain Etherum (comme la plupart des jetons non natifs).

Exemples

Des exemples de tokens sont :

  • natifs (native tokens "coins")
    • bitcoin de Bitcoin
    • ether de Etherum
    • SOL de Solana
    • AVAX de Avalanche
  • security tokens (créances considérées comme des actifs financiers (equity).)
    • RealT pour investir dans des biens immobiliers à la façon d'une SCI
  • utilitaires (utility tokens)
    • CHSB de SwissBorg permet d’augmenter les intérêts de ses cryptomonnaies placées dans le Smart Yield
    • AAVE permet d’obtenir des droits de gouvernance
    • BNB permet de payer les frais de trading sur Binance et autres avantages.
  • non fongibles
    • colored coins sur BTC
    • NFT
