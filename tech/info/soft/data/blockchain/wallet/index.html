Portefeuille.

Motivation

Stocker des tokens.

Analyse

Il existe des portefeuilles :

  • publics, stockés sur les serveurs d'une société. Par exemple:
    • Coinbase
    • Kraken
    • Binance
    • etc.
  • privés, stockés sur un support détenu par le propriétaire (généralement via une application sur ordinateur/téléphone). Par exemple:
    • MetaMask
    • Arianee
    • etc.

Une application de portefeuille permet de transférer des tokens vers une adresse de destination.
