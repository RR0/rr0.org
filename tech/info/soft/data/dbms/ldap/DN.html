Distinguished Name : nom distinct (unique).

Besoin

Décrire une identité dans un certificat X.509.

Analyse

Les noms composants un DN sont :

  • votre nom complet (nom et prénom)
  • le nom de votre département dans votre organisation
  • le nom de votre organisation
  • le nom de votre lieu de résidence
  • le nom de votre province ou état
  • le nom de votre pays

Conception

Un DN est une suite de noms décrivant une identité dans un certificat X.509.

Notes

  • Un DN est souvent représenté sous la forme d'un nom LDAP comme cn=Jérôme Beau, o=Valtech, c=FR
