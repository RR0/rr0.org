Lightweight Directory Access Protocol : protocole standard d'annuaire.

Motivation

Définir un protocole standard d'accès à un annuaire.

Caractéristiques

  • Standard défini par la RFC 2251 ;
  • Hiérarchique ;
  • Un annuaire est accédé depuis Java via JNDI ;

Limitations

  • Non transactionnel ;
  • Peu adapté aux mises à jour fréquentes (optimisé pour la lecture) ;
  • Peu adapté aux relations ;
  • Clustering non standard ;

Exemples

Des exemples d'annuaires LDAP sont :

  • Netscape Directory Server
  • Lotus Domino
