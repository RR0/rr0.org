Shadow DOM :
DOM caché
.

Besoin

Définir un DOM décrivant la structure interne d'un élément personnalisé.

Conception

Le style d'un DOM caché peut être encapsulé, isolé du style de la page.

Implémentation

Notes

Le DOM caché est souvent constitué de templates.
