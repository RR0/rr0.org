Americain Standard Code for Information Interchange.

Motivation

Normaliser la représentation de données binaires en caractères

Organisation

Les caractères ASCII sont codés sur 7 bits (de 0 à 127).

Notes

  • Conçu pour les télescripteurs ;
  • Une table "ASCII étendu" code les caractères sur 8 bits (0 à 255). Elle inclut la table ASCII standard (0 à 127) plus une série de caractères "semi-graphiques" variant selon les OS et pages de codes internationales.
  • Peuvent être obtenus en tapant leur numéro décimal sur le clavier numérique tout en maintenant la touche Alt.

Limitations

  • La plupart des codes non imprimables (< 32) ne sont plus utilisés aujourd'hui, à l'exception de Back Space, horizontal TABulation, ESCape.
