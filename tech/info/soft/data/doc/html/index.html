Hyper-Text Markup Language
:
langage de balises pour l'hypertexte
.

Besoin

Définir un language standard de description de documents sur le Web.

Analyse

HTML est un langage permettant de décrire :

  • le contenu : les données à présenter
  • la présentation : comment les données sont affichées (couleurs, fontes, casse, justification, etc.)
  • le format de saisie d'informations saisies par le client (formulaires)

Conception

Un document HTML est un document texte encodable en UTF-8, ISO-8859-1 ou autre contenant une structure hiérarchique de balises (tags) imbriquées, dont :

  • Un en-tête (head)
  • Un corps (body) de document

Ces balises peuvent toutefois en référencer d'autres à travers certains de leurs attributs.

Cette structure est interprétée par un navigateur Web pour en constituer un DOM.

Implémentation

Exemples

Un exemple de document HTML est :


<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "https://www.w3.org/TR/html4/strict.dtd">
<html>
  <head>
    <title>Ma page</title>
  </head>
  <body>
    <p>Texte de la page <strong>en gras</strong> avec un <em><a href="Lien.html">lien
      en italique</a></em>.</p>
  <body>
</html>

  • Langage (ensemble des balises) limité car figé dans une version
