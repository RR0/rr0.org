Custom elements
:
éléments personnalisés
.

Besoin

Étendre la grammaire d'un langage de balises (HTML).

Analyse

Conception

Un élément personnalisé peut nécessiter :

  • un DOM caché (shadow DOM) pour représenter sa structure interne

Implémentation

La liste des éléments personnalisés est disponible dans window.customElements

Notes

Les éléments personnalisés sont un des constituants des composants Web.
