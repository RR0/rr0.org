head
:
En-tête
.

Besoin

Décrire un document HTML.

Analyse

Conception

  • titre (title) du document
  • metadonnées à propos du document
  • liens relatifs au document

Implémentation

Une balise head doit se placer dans la balise html. Elle doit être de la forme : 

<head>
  contenu d'en-tête
</head>
