Link
:
lien
.

Besoin

Décrire un lien inhérent entre le document HTML et d'autres documents.

Analyse

Conception

  • rel (relation) : type de lien
  • href : valeur du lien

Implémentation

Une balise link doit se placer dans la balise head. Elle doit être de la forme : 

<link rel="type de relation" href="valeur du lien"/>

Parmi les types de liens possibles on trouve :

Type de lien "Link types", HTML4, W3C href
stylesheet Feuille de style du document
prev Page précédente du document
next Page suivante du document
canonical Version la plus typique du document
contents Table des matières du document
start Premier document
index Index du document
help Aide sur le document
alternate Version alternative (langue, format, etc.) du document
meta
