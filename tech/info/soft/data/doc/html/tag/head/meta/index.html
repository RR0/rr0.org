Metadata
:
Meta-données
.

Besoin

Fournir des informations supplémentaires à la description d'un document HTML.

Analyse

Conception

  • name : nom d'une propriété meta
  • charset : jeu de caractères

Implémentation

Une balise meta doit se placer dans la balise head. Elle doit être de la forme : 

<meta name="nom propriété" content="valeur propriété"/>

Propriétés meta

Nom Contenu
description Description du contenu du document
keywords Mots-clés relatifs au document
author Auteur du document
viewport Fenêtre de visualisation
