Viewport
:
Fenêtre de visualisation
.

Besoin

Fournir des metadonnées relative à la fenêtre de visualisation d'une page web.

Analyse

Conception

La métadonnée viewport acceptant elle-même divers paramétrages, elle décompose propose d'inclure une série de clés/valeurs dans sa propre valeur.

  • clé : nom d'une propriété viewport
  • valeur : valeur de la propriété

Implémentation

Une directive viewport est une balise meta dont la clé vaut viewport et la valeur une série de clés-valeurs : 

<meta name="viewport" content="clé1=valeur1,clé2=valeur2"/>

Clés/valeurs viewport

Clé Valeur Commentaire
width device-width Largeur du document = largeur de l'appareil (en pixels CSS)
height device-height Hauteur du document = hauteur de l'appareil (en pixels CSS)
initial-scale 1.0 Niveau de zoom initial
minimum-scale 0.5 Niveau de zoom minimum
maximum-scale 3.0 Niveau de zoom maximum Fixer un tel maximum n'est pas recommandé car cela peut empêcher la lecture par des personnes ayant des déficiences visuelles.
user-scalable no Interdire le zoom Fixer une telle valeur n'est pas recommandé car cela peut empêcher la lecture par des personnes ayant des déficiences visuelles.

Exemples

Page web compatible mobile

S'assurer qu'une page web ne sera pas dézoomée sur des appareils à l'écran trop petit : : 

<meta name="viewport" content="width=device-width, initial-scale=1.0"/>
