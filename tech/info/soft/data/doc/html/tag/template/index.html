Template
:
Modèle de structure
.

Besoin

Représenter une structure de contenu référençable (et donc répétable).

Conception

Les templates sont des balises HTML dont le contenu constitue un modèle de structure référençable.

Implémentation

Un modèle de structure se définit par la balise template : 

<template>Contenu du template</template>

Notes

  • Les templates peuvent être utilisé pour définir le DOM caché d'éléments personnalisés.
