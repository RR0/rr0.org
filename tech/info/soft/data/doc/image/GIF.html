Graphics Interchange Format : format d'échange de graphiques.

Besoin

Fournir un format d'image à grande compression.

Conception

GIF utilise l'algorithme de compression LZW.

Code les nombres en Big Endian.

Notes

  • Le brevet revendiqué sur le format GIF a motivé le développement du format PNG.
  • Créé en environnement PC.
  • A popularisé LZW
