Scalable Vector Graphics : graphiques vectoriels redimensionnables.

Besoin

Format d'image et animations 2D précis et redimensionnables adaptés au Web.

Analyse

SVG supporte divers types d'objets graphiques :

  • Les formes vectorielles (composées de lignes droites et courbes)
  • Les images
  • Le texte
  • les objets externes

Ces objets graphiques peuvent être groupés, on peut leur appliquer des styles, les transformer et composés d'objets préalablement rendus. SVG supporte les transformations embarquées, des tracés de clipping, les masques alpha, les effets de filtre et les modèles d'objets.

Les dessins SVG peuvent être interactifs et dynamiques. Les animations peuvent être définies et déclenchées déclarativement (c.-à-d., en incluant des éléments d'animation dans le contenu SVG) ou par script.

Conception

SVG définit un schéma XML pouvant contenir :

  • une <viewBox> contenant une liste de valeur définissant un système de coordonnées :
    • minX
    • minY
    • largeur
    • hauteur
  • des contextes graphiques <g>
    • des formes  ayant :
      • un nom avec des attributs spécifiques :
        • <circ> avec les attributs <cx>, <cy>, <r>
        • <rect> avec les attributs <x>, <y>, <width>, <height>
        • <line>
        • <path>
      • des attributs communs :
        • <stroke> pour la couleur de trait
        • <stroke-width> pour la largeur de trait
        • <stroke-dasharray> pour les largeurs de traits et espaces entre traits
        • <stroke-dashoffset> pour le décalage des traits
        • <pathLength> pour la longueur du trait
        • <fill> pour la couleur de remplissage
        • <transform> pour la tranformation à appliquer au contexte (rotate (-10) par ex)

Implémentation

Document

Un exemple de document SVG est :


    <?xml version="1.0" standalone="no"?>
    <!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN" "https://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd">

JavaScript

La création de noeuds SVG doit se faire avec le namespace correspondant. Par exemple :


      const circle = document.createElementNS("http://www.w3.org/2000/svg", "circle");
