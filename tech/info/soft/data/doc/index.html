File.

Besoin

Représenter un ensemble de données stocké sur support persistant (disque, mémoire flash, etc.).

Analyse

Un fichier est caractérisé par :

  • une localisation
  • un nom
  • une extension, censée dénoter son type/format.

Conception

Afin de pouvoir interpréter le contenu du fichier selon un format donné, un lecteur pourra se baser sur :

  • son extension
  • son en-tête, généralement les premiers octets du fichier.
