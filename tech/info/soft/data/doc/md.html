Motivation

Rédaction simplifiée de documents HTML.

Analyse

Markdown propose de rédiger des documents à l'aide de raccourcis syntaxiques pour les différentes structures HTML :

Markdown HTML Description
*Texte* <h1>Titre</h1> Titre 1
# Titre <h1>Titre</h1> Titre 1
## Sous-titre <h2>Sous-titre</h2> Titre 2
### Sous-sous-titre <h3>Sous-sous-titre</h3> Titre 3
#### Sous-sous-titre <h4>Sous-sous-titre</h4> Titre 4
* Element 1
* Element 2		 <ul>
  <li>Element 1</li>
  <li>Element 2</li>
</ul> 		Liste non ordonnée
1. Element 1
1. Element 2		 <ol>
  <li>Element 1</li>
  <li>Element 2</li>
</ol> 		Liste ordonnée
