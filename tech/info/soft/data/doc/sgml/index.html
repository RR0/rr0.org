Standard Generalized Markup Language
langage de balises généraliste standard

Besoin

Définir un meta-langage pour définir la structure de documents.

Structure

SGML est un langage composé de balises.

Caractéristiques

  • HTML et XML sont des sous-ensembles de SGML (définis par des DTD SGML), adaptés aux contraintes du Web

Limitations

  • Complexe
