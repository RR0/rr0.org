Donnée informatique.

Motivation

Représenter une information stockable dans un ordinateur.

Analyse

Une donnée est représentée avec un format dans une mémoire.

Conception

Une donnée peut être stockée :

  • en RAM
  • en ROM
  • sur un fichier

Exemples

  • Polices de caractères
  • DBMS
  • Documents
  • Blockchain
