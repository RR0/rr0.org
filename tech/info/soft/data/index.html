Donnée informatique.

Motivation

Représenter une information stockable dans un ordinateur.

Analyse

Une donnée est représentée avec un format dans un support de stockage.

Conception

Une donnée peut être stockée :

  • en RAM
  • en ROM
  • sur un support persistant (disque, mémoire flash, etc.)

Exemples

  • Polices de caractères
  • DBMS
  • Documents
  • Blockchain
