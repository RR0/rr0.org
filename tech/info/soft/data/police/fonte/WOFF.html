Web Open Font Format : Format Libre de Fontes Web.

Besoin

Fournir un format de fonte de caractères libre d'utilisation et adapté au Web.

Analyse

Une police de caractères SFNT (TrueType, OpenType, Open Font Format) doit être convertie au format WOFF via un outil de codage adapté "@FONT-FACE Generator", Font Squirrel.

Conception

Le format utilise la compression ZLib, ce qui permet d'avoir une réduction de taille de fichier de plus de 40 % "WOFF File Format".

Implémentation

Propriétés 1 (2010)

Exemples

Un exemple de feuille d'utilisation de fonte WOFF est :

<!--_woff.css 
 @font-face { 
     font-family: Woff; 
     src:
    url('font/Starcraft.woff'); 
 } 
 .woff {
    
     font: 30px Woff; 
 } 
 -->
