Font.

Besoin

Définir un variante de police.

Analyse

Les variations d'une police sont caractérisées par :

  • sa casse : majuscules ou minuscules
  • sa graisse : non-gras par défaut (ou hérité)
  • sa taille : 1em (16 px) par défaut

Exemples

Un exemple de feuille de style CSS est :

<style type="text/css">
 
 .mainTitle
        {  // class
 font-family: Verdana, Arial,
        Helvetica, sans-serif; 
 font-size: larger
 }
 
 h1 { 

        font-family: "Bookman Old
  Style"; 
 font-size: x-large; 
 font-style: normal; 
 font-weight: normal; 

            font-variant: normal; 
 }
 
 p { 
 font-size: small; 

            font-style: normal; 
 color: #000000; 

            font-family: "Bookman
  Old Style"; 
 }
 
 address { 
 font-family: "Courier
  New", Courier, mono; 
 font-size: small; 
 }
 </style>

Java :

someGraphics.setFont(new java.awt.Font("Courier", Font.BOLD, 36));

Un exemple de page Web (HTML) utilisant une feuille de style est :

     <h1>Ce titre 1 est bleu</h1>

  green">Alors que ce paragraphe est vert.
