Apprentissage profond : DNN.

Motivation

Reconnaissance d'abstractions.

Analyse

L'apprentissage profond consiste à apprendre des concepts très concrets (des pixels par exemple) puis remonter réutiliser cet apprentissage comme entrées des couches au-dessus afin de dégager.

Conception

Une manière de représenter cette profondeur est celle d'un réseau de neurones multi-couches. Ces réseaux peuvent eux-mêmes être connectés :

  • pleinement, ce qui a l'inconvénient d'être prompt au surapprentissage
  • hiérarchiquement : CNN.

Exemples

  • Applications
    • IR (art style transfert learning, deepfake DeepSketch par exemple)
    • Reconnaissance vocale
    • NLP
  • Bibliothèques
    • PyTorch
    • Caffe2
    • WebDNN pour les exécuter (via GPU) dans un navigateur
  • Frameworks
    • ConvNetJS - Réseaux neuronaux convolutifs dans le navigateur
  • L'apprentissage profond de Google est constitué de 10 à 30 couches.
