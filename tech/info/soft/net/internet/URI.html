Uniform Resource Identifier : identifiant uniforme de ressource.

Besoin

Définir un moyen standard de trouver une ressource (fichier, page Web, image, etc.) sur Internet

Conception

Une URI est composée de :

  • une partie spécifique au schéma ;
  • (optionnel) # suivi d'une référence de fragment ;

Une URI peut être :

  • absolue (spécifiant un schéma, suivi de : ou relative
  • hiérarchique (continuant par /) ou opaque

Exemples

  • absolues
    • https:
    • mailto:
    • news:
  • hiérarchiques
    • https://www.javarome.net
    • docs/guide/collections/designfaq.html#28
    • ../../../demo/jfc/SwingSet2/src/SwingSet2.java
    • file:///~/calendar
  • opaque
    • mailto:javarome@javarome.net
    • news:comp.lang.java
    • urn:isbn:096139210x

Notes

  • Défini par la RFC 2396
  • Les URL sont des URI indiquant comment localiser la ressource recherchée
  • Les URN sont des URI ne spécifiant pas comment localiser la ressource recherchée
