Post Office Protocol : protocole de courrier postal.

Motivation

Recevoir des courriers électroniques (e-mails).

Organisation

POP est basé sur TCP/IP.

Notes

  • La version couramment utilisée est POP3.
  • Utilise le port 110.

Exemple

Des exemples de serveurs POP3 sont :

  • Jakarta James
  • Microsoft Exchange

Un exemple de session POP sur MS Exchange est :

+OK Microsoft Exchange POP3 Server version 5.5.2653.23 prêt
USER mondomaine\javarome\javarome
+OK
PASS password
+OK
