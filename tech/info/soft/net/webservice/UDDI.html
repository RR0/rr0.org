U. D. D. I.

Universal Description, Discovery and Integration : description, découverte et intégration universels.

Besoin

Mettre en place un annuaire public (Internet) des services Web publiés par les entreprises.

Principes

L'annuaire UDDI distribué sur Internet est pensé pour comporter trois types d'informations :

  • pages blanches : description des entreprises (contacts, adresse, raison sociale, etc.)
  • pages jaunes : description des services (métier de l'entreprise, services proposés)
  • pages vertes : description technique (URL d'accès, protocoles utilisés, interfaces implémentées, etc.)

Notes

  • Géré par le W3C
  • Beaucoup plus utilisé en interne (EAI) que sur Internet.
  • Encapsulé derrière WS-Inspection qui regroupe l'ensemble des informations sur un service Web (WSDL, UDDI, autres).

Limitations

  • Pas de mécanisme pour évaluer la qualité de service
  • Court-circuitage des relations humaines peu adapté aux habitudes des entreprise
  • Fournit une description technique des services redondante avec WSDL
