Syntaxically Awesome Style Sheets : feuilles de style syntaxiquement géniales

Besoin

Etendre la syntaxe CSS.

Analyse

Sass est un langage compatible avec CSS, avec des concepts additionnels de :

  • variablesAvant leur introduction dans CSS
  • inpm rebuild node-sassmbrication (nesting) pour éviter les répétitions
  • imports (partials) d'autres fichiers Sass
  • déclarations réutilisables (mixins)
  • héritage (inheritance) de classes CSS
  • calculs via le support d'opérateurs Avant le support du calc() CSS

Conception

Sass est à l'origine une application Ruby, mais est utilisable par d'autre langages via la bibliothèque native LibSass (écrite en C++).

Variables

Une variable Sass est définie sous la forme $myVariable.
