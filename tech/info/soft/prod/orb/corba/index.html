Common ORB Architecture : architecture d'ORB commune.

Besoin

Standardiser le communication entre objets distants :

  • indépendamment du langage
  • indépendamment de la plate-forme

Analyse

CORBA est composé de :

  • un ORB utilisant
    • un protocole dérivé de GIOP, comme :
      • IIOP
    • IDL (Interface Definition Language) pour spécifier les interfaces entre objets communicants.
  • des services horizontaux au dessus de cet ORB
  • des dispositifs (facilities) verticaux au-dessus de ces services

Implémentation

JavaIDL
