Operating System : Système d'exploitation (SE)

Besoin

Gérer l'ordinateur, les logiciels qui s'y exécutent, et les interactions avec l'utilisateur.

Conception

Un shell (coque) permet à l'utilisateur d'accéder aux fonctions de l'OS.

Exemples

Des exemples de système d'exploitation sont :

  • DOS
    • MS-DOS
    • IBM-DOS
  • Windows
  • Unix
    • Linux
  • Apple
    • Mac OS
    • iOS
