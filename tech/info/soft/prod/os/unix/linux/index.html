Linus' uniX : l'Unix de Linus.

Besoin

Développer un noyau Unix et Open Source.

Conception

Au sens strict, Linux est le noyau (kernel) du système d'exploitation GNU.

Exemples

Des distributions de Linux sont :

  • Debian - Distribution la plus "libre" (aucun copyright)
  • Ubuntu
  • RedHat - Pour les entreprises
    • Fedora - Pour le grand public
  • Mandrake
