Unix shells : coques d'un OS Unix

Besoin

IHM permettant d'accéder aux fonctions d'un OS.

Conception

Un invite de commande (prompt) de saisir diverses commandes avec des arguments variables.

Exemples

Des exemples de shells Unix sont :

  • Bourne Again SHell (bash)
  • C Shell (csh)
  • Tenex C Shell (tcsh)
  • Korn SH (ksh)
  • Z Shell (zsh)

Notes

  • On peut connaître son shell via echo $0.
  • On peut changer son shell via chsh. Par exemple chsh -s /bin/ksh.
  • MacOS utilisait tcsh jusqu'à sa version 10.3 (Panther) où il inclut bash.
