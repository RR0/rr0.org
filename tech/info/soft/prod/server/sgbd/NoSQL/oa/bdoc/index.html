Base de Donnée Orientée Colonnes.

Besoin

BDOA adaptable à divers cas.

Analyse

Une BDOC est :

  • un espace de clés (keyspace) référençant :
    • une famille de colonnes, chaque colonne pouvant être :
      • simple (une seule valeur) ou multiple (super colonne, regroupant plusieurs colonnes)
      • statique (structure fixe) ou dynamique (variable selon les clés)

Produits

  • HBase
