Base de Données Orientées Agrégats.

Besoin

Analyse

Base NoSQL pour stocker des grappes (agrégats) de données.

Conception

De par leur conception, les BDOA privilégient un sens de navigation des données (de la clé vers la donnée).

Exemples

Des exemples de BDOA sont les :

  • ECV
  • BDOD
  • BDOC
