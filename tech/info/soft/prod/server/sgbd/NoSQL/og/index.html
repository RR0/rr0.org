Base de Données Orientées Graphes.

Besoin

Effectuer des opérations sur des graphes persistants de manière performante.

Analyse

Une BDOG permet par exemple d'effectuer efficacement :

  • le parcours entier d'un graphes ;
  • la recherche d'un chemin entre 2 noeuds vérifiant certaines contraintes.

Conception

Les transactions des BDOG sont généralement ACID.

Exemples

Produits

Des exemples de BDOG sont :

  • Neo4J
  • Infinite Graph
  • OrientDB

Cas d'utilisation

  • Réseaux sociaux.
  • Réseaux géographiques.

Notes

  • Type de base NoSQL.
