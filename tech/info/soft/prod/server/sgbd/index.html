Système de Gestion de Base de Données.

Motivation

  • Stocker des données pour les traiter (recherche, mise à jour).

Exemples

Des exemples de SGBD sont :

  • relationnels
  • objet
  • hiérarchiques (SGBDH), selon une organisation parent/enfants
  • réseau : hiérarchique avec n parents (CODASYL par exemple)
  • NoSQL

Notes

  • La motivation des SGBD est par opposition aux systèmes de fichiers, qui ne stoquent que des données brutes, laissant aux applications qui y accèdent la connaissance de leur structure.
