Object DBMS : SGBD Objet (SGBDO).

Besoin

Stocker des objets persistants.

Conception

Les SGBDO stockent l'état des objets, y compris leurs relations (pointeurs sur d'autres objets). Ils permettent également d'exécuter le code des objets persistants.

Notes

  • Plus performants pour la navigation entre objets, grâce à l'utilisation de pointeurs directs sur les instances (pas besoin de recherche de clé comme pour les SGBDR).

Exemples

Des exemples de SGBDO sont :

  • Versant
  • Ardent
  • Object Design
  • GemStone
  • Objectivity
  • Poet Software

Notes

  • Les SGBDO sont des exemples d'EIS.
  • Ils sont typiquement accédés depuis Java via JDO.
