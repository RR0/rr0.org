Besoin

Fournir un SGBDR pour les sites Web Open Source.

Analyse

MySQL supporte divers moteurs de persistance :

  • InnoDB, supportant les transactions ACID et les contraintes d'intégrité référentielle (clés étrangères) via un verrouillage au niveau ligne.
  • MyISAM, ne supportant pas les transactions mais plus performant
  • BDB (Berkeley DataBase), supportant les transactions

La version standard (GPL) de MySQL est déclinée en versions :

  • Pro (licence commerciale)
  • Max, ajoutant des fonctionnalités avancées (BDB, SSL, répartition de tables sur plusieurs fichiers)
  • Classic (licence commerciale) sans InnoDB.

Notes

  • Largement utilisé sur le Web (ce qui a orienté ses fonctionnalités dans ce sens), proposé par la plupart des hébergeurs gratuits, au côté de PHP.
  • Aujourd'hui de plus en plus utilisé dans d'autres contextes (entreprise).

Limitations

  • Ne supporte pas SQL dans sa totalité
    • pas de requêtes imbriquées (select key from table where key in (select otherKey from otherTable)), qui sont pourtant le fondement du terme SQL (Structured QL).
  • Pas de vues
  • Il faut choisir entre une version pour la "performance" et une pour les "fonctionnalités" (XA notamment)
