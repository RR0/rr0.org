Besoin

Reproduire l'environnement de développement pour une version donnée du logiciel développé.

Analyse

La gestion de configuration nécessite :

  • une gestion des versions du logiciel développé ;
  • une gestion de l'environnement logiciel
    • OS si plusieurs plates-formes de développement doivent être supportées
    • dépendances (third party libraries) spécifiques à un OS ou non.

Conception

Gestion de version

La gestion de version se fait généralement via un VCS.

Gestion de l'environnement

La gestion de l'environnement se fait généralement via un outil de virtualisation.
