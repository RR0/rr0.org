Git est un VCS.

Conception

Git est un VCS décentralisé (chaque utilisateur dispose de l'ensemble du repository, il n'y a pas de rôle de client/serveur), ce qui permet des opérations rapides (effectuées sur le repository local, sans besoin d'appels réseau). Il utilise sa propre base de données spécifique. Rangadurai, Kaushik: "Git Internals - How Git works ", LinkedIn, 2015-01-22
