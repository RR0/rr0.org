Besoin

Fournir un service réseau.

Analyse

Un serveur réalise une tâche suite à la requête d'un client donné.

Conception

Un serveur constitue un élément d'une architecture C/S.

Exemples

Il existe divers types de serveurs :

  • un serveur Web fournit des documents Web (pages HTML, images, sons, documents PDF, etc.)
  • un serveur applicatif fournit des services (rendus par les applications qu'il héberge)
  • un serveur de gestion de base de données (SGBD) fournit ou modifie les données qu'il stocke
  • un serveur proxy relaie des requêtes réseaux
  • etc.
