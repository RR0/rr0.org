Hyper-Text Transfer Protocol : protocole de transfert d'hyper-texte.

Besoin

Définir un protocole réseau standard pour l'échange de documents sur le Web (pages HTML).

Conception

HTTP est un protocole de l'Internet, basé sur TCP/IP :

Session
HTTP
Transport
TCP
Réseau
IP

Une trame HTTP est échangée entre un client HTTP (navigateur Web) et un serveur HTTP (serveur Web).

Elle est constituée de :

  • Un en-tête (header) constitué de :
    • Une commande
    • Une URI de ressource à obtenir (monChemin/maResource)
    • Des champs d'en-tête
  • un saut de ligne
  • Des données éventuelles (POST, upload)

Le serveur répond un code HTTP.

Exemples

Un exemple de requête HTTP 1.1 est :

POST javarome/search.html
Content-type: application-ext/form-data
Content-length: 40
Accept-language: FR/fr, EN/uk

query=HTTP&query=protocol&result=display

Notes

  • Les serveurs HTTP sont également appelés serveurs Web
  • Des clients HTTP typiques sont les navigateurs Web
  • Une ressource HTTP sur Internet est localisée via une URL HTTP
  • Les serveurs applicatifs sont souvent également des serveurs HTTP
  • Utilise les types MIME
  • Cookies
  • chunked encoding [1.1]
  • Respect du TTL des DNS [1.1]
  • Diverses extensions existent, comme WebDAV.

Limitations

  • Basé sur les protocoles de courrier Internet (SMTP, MIME) et donc long à décoder, verbeux, découverte de la longueur à la fin de la lecture.
  • Sans état : Aucun état n'est conservé entre deux requêtes/réponses HTTP. Ceci est généralement contourné par :
    • les cookies HTTP
    • la réécriture d'URL (URL rewriting)
  • Déconnecté : ne traite qu'une seule URL par connection TCP. Une fois la réponse à la requête envoyée, la connexion est théoriquement terminée (close). Des contournement sont l'ouverture de plusieurs connexions simultanées (4 par défaut pour Netscape).
    • des alternatives pour faire du push sont :
      • les connexions persistantes de HTTP 1.1
      • les connexions en attente (pending)
      • le rechargement automatisé (refresh après expiration)
