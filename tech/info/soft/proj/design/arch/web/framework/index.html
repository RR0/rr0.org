Cadre d'applications web.

Besoin

Simplifier le développement d'applications web.

Analyse

Les frameworks web peuvent fournir des fonctionnalités telles que :

  • templates incluant :
    • des parties variables
    • des structures de contrôle (if, else)
    • des structures de répétition (boucle)
  • une réactivité des templates
  • un routeur de pages/outlets web
  • une internationalisation des messages

Conception

Comme tout framework, les frameworks web définissent un contrat que les composants web doivent respecter.

Exemples

client global natif
Angular NestJS, Angular Universal
React NextJS React native
Svelte SvelteKit Svelte Native
Vue Nuxt Vue Native

Certains meta-frameworks (Meteor par exemple) sont capables d'intégrer d'autres frameworks.
