Besoin

Garantir une instance unique.

Analyse

Modèle de conception.

Implémentation

public static MySingletonClass getInstance() {
if (mySingleton == null) {
synchronized (MySingletonClass.class) {
mySingleton = new MySingletonClass();
}
}
return mySingleton;
}
