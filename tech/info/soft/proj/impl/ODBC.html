Open DataBase Connectivity.

Motivation

  • Fournir une API standard d'accès aux SGBDR.

Organisation

L'API ODBC est basée sur les spécifications CLI (Call-Level Interface) du X/Open et de ISO/IEC pour les API de SGBD et utilise SQL comme langage d'accès à la base.

Notes

  • Spécifié par Microsoft en 1991.
  • Très largement supporté avec plus de 170 pilotes existants.
  • Intégré à Windows en 1992.
  • Le pilote JDBC de type 1 (JDBC-ODBC bridge) existe pour accéder aux bases de données depuis Java via ODBC.

Limitations

  • Faible performance.

Exemples

ODBC est disponible sur :

  • Windows (1992)
  • AIX
