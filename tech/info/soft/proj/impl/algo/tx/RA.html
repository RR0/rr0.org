Resource Adapter : Adaptateur de ressources.

Besoin

  • Permettre l'accès à un RM.

Analyse

En tant qu'adaptateurs, les RA ont :

  • une API standard
  • une implémentation spécifique au RM

Conception

Un standard de RA est JCA.

Exemples

Des exemples de RA sont :

  • un pilote JDBC pour un RM SGBDR
  • un pilote ODMG pour un RM SGBDO
  • une bibliothèque JRFC pour un RM SAP R/3
