Resource Manager : gestionnaire de ressources.

Besoin

  • Gérer des ressources dans le cadre d'une transaction globale.

Analyse

Dans le cadre d'une transaction, un RM reçoit plusieurs ordres d'opérations.

Conception

Les RM sont accessibles via des RA.

Afin de pouvoir s'intégrer dans une transaction globale, ils implémentent également une API de Resource Transactionnelle utilisée par les TM.

Exemples

Des exemples de RM sont :

  • SGBDR
  • TPM
  • SGBDO

Limitations

  • Tous les RM ne supportent pas XA
  • Tous les RM ne supportent pas tous les niveaux d'isolation
  • Le niveau d'isolation par défaut n'est pas le même pour tous les RM
