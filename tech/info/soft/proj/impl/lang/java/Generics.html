Types génériques ou paramétrés (parameterized types): structures de données dont un ou plusieurs types peuvent être paramétrés.

Besoin

  • Augmenter
    • la robustesse du code
    • la lisibilité du code

Analyse

Les types génériques peuvent être paramétrés par d'autres types.

Notes

  • Bien qu'apparaissant semblables aux mécanisme de template du C++, les génériques de Java :
    • ne génèrent pas une nouvelle classe pour chaque spécialisation
    • ne permettent pas de "métaprogrammation" par modèle (template) puisqu'un typage fort est préservé

Implémentation

Déclaration

Un type générique est postfixé par un (ou plusieurs) marqueur de généricité symbolique :

class MyGenericType<T1,T2,T3> {
...
}

Instanciation

MyGenericType<String,Integer,OtherType> = new MyGenericType<String,Integer,OtherType>();
