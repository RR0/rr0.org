C++ = Successeur du C.

Motivation

Améliorer le C pour supporter :

  • la programmation OO
  • l'abstraction de données
  • la programmation générique

Principes

Basé sur le C + préprocesseur.

Notes

  • Créé par Bjarne Stroustrup en 1983.

Exemples

Un exemple de code C++ est :

class A {
  virtual void foobar (char*);
};

class B : public A {
  virtual void foobar (char*);
};
