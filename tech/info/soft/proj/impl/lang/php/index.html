Pretty Hypertext Preprocessor : joli pré-processeur pour hypertexte.

Motivation

Langage de script pour le Web

Conception

Comme les JSP, PHP insère son code dans du HTML. De la même manière, il nécessite un module lié au serveur Web chargé d'interpréter les scripts PHP correspondant à des URL.

Exemple

Code

Un exemple de code PHP est :

<html>
<head>
<title>Exemple PHP</title>
</head>
<body>
<?php echo "Bonjour, je suis un script PHP"; ?>
</body>
</html>
