Binary Search Tree (BST) : Arbre Binaire de Recherche (ABR).

Besoin

Stocker un ensemble trié de données retrouvables en O(n log(n))

Analyse

Un ABR est un arbre binaire dont les nœuds sont organisés de manière que :

  • Tout nœud fils gauche a une clé inférieure à son père.
  • Tout nœud fils droit a une clé supérieure à son père.
