Virtual Private Network : Réseau Privé Virtuel (RPV).

Besoin

Réseau privé confidentiel.

Analyse

Un VPN permet de créer un réseau logique indépendant du réseau physique :

  • partagé
  • pair à pair (peer-to-peer)

Conception

Passerelle

Tunnel

La confidentialité est assurée par la cryptographie.

SSL

Ouverture des sessions HTTPS (SSL/TLS) depuis un client Web typiquement.

IPSec

Un logiciel client permet la création d'un tunnel utilisant divers protocoles (SSH, SMTP, IMAP, etc.)

Exemples

Usages

Travail à distance

Cloud computing

Implémentations

  • IPSec
  • OpenVPN
  • WireGuard
