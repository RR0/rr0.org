Uniform Resource Locator : localisateur uniforme de ressource.

Besoin

Définir un moyen standard de trouver une ressource (fichier, page Web, image, etc.) sur Internet

Conception

Une URL est une URI composée de :

  • un protocole
  • un nom de machine
  • (optionnel) un port précédé de : si l'on veut accéder un autre port que celui par défaut du procotole
  • un nom de domaine
  • un chemin d'accès à la ressource sur la machine
  • (optionnel) un fragment précédé de #

Notes

  • Étendu par la RFC 2732
  • Les caractères d'une URL peuvent être :
    • des caractères alphanumériques [0-9a-zA-Z]
    • les caractères spéciaux et les caractères réservés à leurs places prévues, sinon codés en % suivit du code ISO-Latin hexa du caractère. Il s'agit de (codage obligatoire en gras, sinon recommandé) :
      Caractère Codage URL
      %20 ou +
      %22
      # %23
      $ %24
      % %25
      & %26
      + %2B
      , %2C
      / %2F
      : %3A
      ; %3B
      < %3C
      = %3D
      > %3E
      ? %3F
      @ %40
      [ %5B
      \ %5C
      ] %5D
      ^ %5E
      ` %60
      { %7B
      | %7C
      } %7D
      ~ %7E
