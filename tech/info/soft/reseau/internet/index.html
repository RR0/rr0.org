Réseau (net) interconnecté.

Besoin

Interconnecter des réseaux.

Analyse

L'internet est notamment constitué de :

  • clients
  • serveurs (origin servers)
  • passerelles (gateways)
  • proxies

reliés par réseau.

Conception

Techniques

Le protocole de l'Internet est IP (Internet Protocol).

Divers protocoles ou applications se basent sur IP :

Application HTTP FTP DNS Traceroute Ping
Transport
TCP
 UDP ICMP
Réseau
IP

Notes

  • Initié en 1972 avec le Défi Internet (Internet challenge) par l'ARPA qui identifie le besoin d'interconnexions de réseaux tandis que le nombre de technologies différentes croit. L'ARPA finance les études, et entre 1977/1979 les protocoles TCP/IP prennent leur forme définitive. De 1980/1983 tous les réseaux raccordés à ARPANet. sont convertis à TCP/IP. En 1984 l'ARPA stoppe son financement et en 1986 l'ARPANet est séparé en Milnet et NSFNet. En 1989 l'ARPANet est arrêté. En 1994 le réseau n'est plus opéré par la NSF, et en 1995 NSF Net est arrêté.
  • L'application la plus connue de l'internet est le Web.
  • MIME standardise les types de données échangées sur Internet.
  • Les bonnes pratiques de comportement sur Internet sont décrites dans La Netiquette (RFC 1855).
  • En 1992 est créé l'ISOC, le gouvernement de l'Internet.
  • Parallèlement, le DARPA finance l'Université de Berkeley et TCP/IP est intégré dans Unix, ce qui lui vaut une très large expansion.
  • La taille de l'Internet (nombres de postes et population des utilisateurs) double chaque année.
  • Les standard de l'Internet sont définis par les RFC.
  • Aujourd'hui on peut se connecter à Internet via :
    • des liaisons spécialisées (ligne louée auprès de FT, 1200 bps à 34 Mbps, tarif forfaitaire fonction de la distance)
    • Numéris ou Transpac (connexion RNIS ou X25, 64 Kb/s)
    • le réseau téléphonique (modem ADSL ou non)
    • le câble
    • les connexions non IP
