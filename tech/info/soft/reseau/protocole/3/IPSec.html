IP Security : sécurité de IP.

Besoin

Fournir au niveau IP (réseau) :

  • authentification
  • confidentialité

Conception

IP Sec définit :

  • protocoles de sécurité:
    • Utilisation de l'en-tête d'authentification IPv6 (Authentication Header ou AH)
    • Charge d'encapsulation de sécurité (Encapsulating Security Payload ou ESP)
  • associations de sécurité :
  • gestion des clés manuelle et automatique (IKE)
  • algorithmes d'authentification et de cryptage

Notes

  • supporte IPv4 et IPv6

Voir

  • RFC 2401 : Security Architecture for the Internet Protocol
