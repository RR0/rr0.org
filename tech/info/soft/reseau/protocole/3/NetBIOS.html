Network BIOS : BIOS réseau.

Besoin

Partager des resources sur le réseau.

Analyse

NetBIOS est une couche logicielle utilisée par le système de gestion de réseau MicroSoft (Microsoft Networking).

Conception

NetBIOS est configuré pour faire le lien entre 9 composants de votre PC, situés sur 3 couches réseau : matérielle, transport et services :

Service Client pour Microsoft Networks Partage fichiers et imprimantes pour Microsoft Networks Login Microsoft family
Tous liens avec Transport
Transport TCP/IP NetBEUI IPX / SPX
Tous liens avec Matériel
Matériel Adaptateur pour numérotation téléphonique Interface câble modem / DSL Interface LAN (si disponible)
Vers... Internet Réseau local (si disponible)

Exemples

Des exemples d'utilisation de NetBIOS sont :

  • Partage sur un LAN
    • de fichier(s)
    • d'imprimante(s)

Conception

IPX est un protocole datagramme ou paquet. Il se situe au niveau de la couche réseau et est non connecté.

Limitations

  • Non recommandé pour une connexion Internet (faille de sécurité).

Notes

  • L'accusé réception des paquets est réalisé par SPX (Sequenced Paquet Exchange).
  • D'autres protocoles de Novell sont :
    • Routing Information Protocol (RIP),
    • Service Advertising Protocol (SAP),
    • NetWare Link Services Protocol (NLSP).
