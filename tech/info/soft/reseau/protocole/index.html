  1. Application :
  2. Présentation

Notes

  • Sous Unix, le fichier /etc/protocols fait correspondre un numéro (propre au système) de protocole avec un nom officiel ou des synonymes. Par exemple :
    ip 0 IP
    icmp 1 ICMP
    tcp 6 TCP
    udp 17 UDP
