Access Control List : liste des contrôles d'accès.

Besoin

Définir les permissions d'accès à une ressource.

Conception

Une matrice d'accès (access matrix) définit pour chaque utilisateur ses droits sur un "objet protégé" ou ressource.

Exemples

main.c a.out log.txt Compiler
Vendor read,write read,write write
User read, write read call
Compiler read write write
main.c a.out log.txt Web browser
Web site read,write read,write write write
Web browser read, write read read
Web page read, write read read
Web browser
