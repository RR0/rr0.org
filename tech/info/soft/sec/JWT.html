JSON Web Token : jeton web au format JSON.

Besoin

Conception

Un jeton se compose de :

  • Un en-tête (header) décrivant le jeton au format JSON :
    • typ : Type de jeton ("jwt" ou autre)
    • alg" : Algorithme de signature ("HS512" ou autre)
  • Une charge utile (payload) qui représente les informations transmises par le jeton, au format JSON.
  • Une signature numérique au format header.alg.

Le JWT à transmettre est généré via forme : signature(header + "." + payload, signPrivKey)
