Secure SHell : coquille sécurisée.

Besoin

Fournir des communications sécurisées entre machines sur Internet.

Conception

Protocole.

SSH Version 1 2 Commentaire
Cryptographie à clé publique RSA DSA

Implémentation

Une suite SSH consiste typiquement en 2 ou 3 applications :

  • (toujours) ssh, qui établit et accède une session sécurisée interactive (un invite de commandes) sur une machine distante.
  • (souvent) scp (Secure CoPy), utilisé pour les transferts sécurisés de fichiers d'un point à un autre.
  • (parfois) sftp (Secure FTP), proposant généralement une IHM similaire aux clients FTP, utilisé pour transférer des fichiers de manière sécurisée depuis ou vers une machine distante.

Notes

  • préféré aux protocoles FTP et TelNet qui sont des protocoles historiques non sécurisés.
