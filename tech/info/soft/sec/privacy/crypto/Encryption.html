Encryption. Parfois le terme "chiffrement" est utilisé.

Besoin

Remplacer un message clair par un message codé.

Analyse

Il existe deux grands types de chiffrement :

  • symétrique
  • asymétrique

De plus les chiffrements peuvent s'effectuer :

  • par blocs de données (dont la taille est déterminée)
  • sur des flux de données (dont la taille est indéterminée)

Limitations

  • Un acteur intermédiaire peut toujours essayer de modifier les messages cryptés échangés, alors que ceux-ci sont considérés comme sûrs par le client. Ces messages étant cryptés, les chances de succès d'une telle tentative varient en fonction du protocole utilisé, mais existent belles et bien. On veillera donc à garantir l'intégrité des messages échangés.
